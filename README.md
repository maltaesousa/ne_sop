# ne_sop
Suivi des objets parlementaires

## Build

In terminal:
1. Change current directory to ne_sop\front
2. Run `vite build`

## Run

In terminal:
1. Change current directory to ne_sop\front
2. Run `vite`

## Deploy

cd "back"
Activate the python virtual environnement: ".venv\Scripts\activate"
Inside venv: pip install -r requirements.txt
